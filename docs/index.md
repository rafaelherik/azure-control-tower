Welcome to the Azure Command Tower documentation! Azure Command Tower (azct) is a terminal-based UI for exploring and managing Azure resources.

## What is Azure Command Tower?

Azure Command Tower provides an intuitive terminal interface for navigating your Azure cloud resources. Instead of using the web portal or CLI commands, you can browse subscriptions, resource groups, and resources with a modern TUI (Terminal User Interface).

## Key Features

- **Browse Azure Resources**: Navigate through subscriptions, resource groups, and resources
- **Storage Explorer**: Explore Azure Storage accounts, containers, and blobs
- **Filter & Search**: Quickly find resources using built-in filtering
- **Resource Details**: View detailed information about any Azure resource
- **Modern TUI**: Beautiful terminal interface built with tview
- **Fast & Lightweight**: Native Go application with minimal dependencies

## Quick Start

1. Install azct (see [Installation](getting-started/installation.md))
2. Authenticate with Azure: `az login`
3. Run: `azct`
4. Start exploring!
