# Keyboard Shortcuts

Azure Command Tower provides keyboard shortcuts for efficient navigation and interaction.

## Global Shortcuts

| Key | Action | Description |
|-----|--------|-------------|
| `q` | Quit | Exit Azure Command Tower |
| `/` | Filter | Open filter/search (in table views) |
| `m` | Menu | Open resource type menu |
| `ESC` | Back | Navigate back to previous view |

## Navigation

| Key | Action | Description |
|-----|--------|-------------|
| `↑` / `↓` | Navigate | Move selection up/down |
| `Enter` | Select | Select item or view details |
| `Page Up` / `Page Down` | Scroll | Scroll through long lists |

## View-Specific Actions

### Subscriptions View

| Key | Action |
|-----|--------|
| `Enter` | View resource groups |
| `d` | Show subscription details |

### Resource Groups View

| Key | Action |
|-----|--------|
| `Enter` | View resource types |
| `d` | Show resource group details |

### Resource Types View

| Key | Action |
|-----|--------|
| `Enter` | View resources of selected type |
| `m` | Open resource type menu |

### Resources View

| Key | Action |
|-----|--------|
| `Enter` | View resource details |
| `d` | Show resource details |
| `e` | Explore storage (storage accounts only) |

### Storage Explorer View

| Key | Action |
|-----|--------|
| `Enter` | Open container |
| `d` | Show container details |

### Blobs View

| Key | Action |
|-----|--------|
| `Enter` | Navigate folder or view blob |
| `d` | Show blob details |

### Details View

| Key | Action |
|-----|--------|
| `ESC` | Go back |

## Filter Mode

When in filter mode (activated with `/`):

| Key | Action |
|-----|--------|
| `ESC` | Cancel filter |
| `Enter` | Apply filter |
| Any text | Type to filter |

## Tips

- Most actions are context-sensitive and change based on the current view
- The footer always shows available shortcuts for the current view
- Use `ESC` to navigate back through the hierarchy
- Filter mode works in all table views for quick searching
